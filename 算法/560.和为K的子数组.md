简化版的 [[560.和为K的子数组]]

```java
class Solution {
    public int subarraySum(int[] nums, int k) {
        int result=0;
        int sum=0;
        Map<Integer,Integer> map=new HashMap<>();
        map.put(0,1);
        for(int num:nums){
            sum+=num;
            int count=map.getOrDefault(sum-k,0);
            result+=count;
            int temp=map.getOrDefault(sum,0);
            map.put(sum,temp+1);
        }
        return result;
    }
}
```
